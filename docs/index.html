Methodology

Each speech is compared against randomly selected opponents from within a ±3 year window using Gemini 2.0 Flash. The model is asked which speech takes a more hawkish position on monetary policy — meaning a stronger preference for tighter policy, higher interest rates, or greater concern about inflation risks relative to growth risks.

Speaker names and titles are stripped from speech text before comparison so the model cannot use prior knowledge of a speaker's reputation. Ratings are tracked using TrueSkill (μ=25, σ=8.333), a Bayesian skill rating system. Speeches exit the active pool once their uncertainty (σ) falls below 4.5.

Raw scores are min-max normalized to a 0–100 scale. The era-adjusted score subtracts the quarterly mean, re-centering to 50, to isolate each speaker's disposition from the prevailing policy environment. Speeches about topics unrelated to monetary policy (banknote launches, cyber security, etc.) are excluded.

Inspired by FedLock, an experiment by Joe Weisenthal applying this methodology to Federal Reserve speeches. Source code available on GitHub.
